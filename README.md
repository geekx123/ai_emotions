# ai_emotions

Fun, lightweight computer-vision app that detects human facial emotions in real-time and overlays meme-style filters such as sunglasses, horns, tears, etc.
Built using Python, OpenCV, TensorFlow/Keras, and Streamlit.

Features

--> Real-time emotion detection using a CNN model trained on FER2013 dataset

--> Automatic face detection via OpenCV Haarcascade

--> Interactive web UI made with Streamlit

--> Meme-style overlays applied dynamically on detected faces

--> Fast inference suitable for laptops and mobile browsers

--> Works with image uploads or live webcam feed

Tech Stack

--> Python 3.10+

--> OpenCV – face detection

--> TensorFlow/Keras – CNN emotion classifier

Streamlit – web interface

NumPy & Pandas – preprocessing utilities
